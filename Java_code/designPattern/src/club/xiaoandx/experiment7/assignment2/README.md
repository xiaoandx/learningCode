### 软件设计模式 - 实验七- 实验内容二

#### 试题题干说明：

现假设要开发一个游戏UI系统，分为2个面板，请使用观察者模式，实现更新UI的框架代码，即：主角属性是由服务器获取，然后分发给对应的面板，进行更新。

#### 代码文件说明：

| 序号 | 目录/文件 | 备注                       |
| ---- | --------- | -------------------------- |
| 1    | observer | 观察者文件夹（主要是数据展示面板）|
| 2    | pojo   | 英雄属性实体文件夹   |
| 3    | subject   | 注册中心文件夹（服务器）   |
| 4    | ObserverTest | 观察者模式测试类（运行测试）                   |
